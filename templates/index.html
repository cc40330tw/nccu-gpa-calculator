Tutorial
宗旨
此網站提供政大學生快速計算GPA（包含4.3制、4.0制、Last60、Major等GPA)

＊目前只支援電腦


使用步驟
1. 登入iNCCU
2. 點擊「進入我的全人」
3. 點擊左邊欄位的「課業學習」
4. 下載此頁的html檔案
5. 將此html檔案匯入此網站，並按下Calculate

如何下載html檔案
Mac
  • Safari： Command + S, 格式選擇「網頁原始檔」
  • Chrome： Command + S, 格式選擇「僅限網頁的HTML部分」或「完整的網頁」
